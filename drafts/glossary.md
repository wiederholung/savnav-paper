# SAVNav 论文术语中英对照表 (Glossary)

本文档整理了 SAVNav (Socially-Aware Audio Visual Navigation) 项目及论文中的核心术语、缩写及其标准学术中文翻译，旨在统一论文写作与系统开发中的表达。

## 1. 核心概念与缩略语 (Acronyms & Core Concepts)

| 英文/缩写 (En)                                      | 中文翻译 (Cn)                 | 定义/备注                                                                      |
| :-------------------------------------------------- | :---------------------------- | :----------------------------------------------------------------------------- |
| **SAVNav** (Socially-Aware Audio-Visual Navigation) | **社会感知视听导航**          | 项目名称。指结合视觉与听觉模态，并遵循社会规范（如避让人群）的机器人导航框架。 |
| **Non-Line-of-Sight (NLOS)**                        | **非视距 / 视线外**           | 指视线被障碍物遮挡的区域。本文核心解决该区域内的潜在风险。                     |
| **Line-of-Sight (LOS)**                             | **视距 / 视线内**             | 传感器（相机）可直接观测到的区域。                                             |
| **Corner Surprise**                                 | **转角突发状况 / 拐角惊吓**   | 特指动态行人突然从拐角盲区出现的场景，是 NLOS 导航的主要挑战。                 |
| **Social Navigation**                               | **社交导航 / 社会化导航**     | 强调机器人在人流中导航时需遵循人类社交规范（如保持距离、不惊吓行人）。         |
| **Audio-Visual Navigation (AV-Nav)**                | **视听导航**                  | 同时利用视觉（图像/深度）和听觉（声音信号）进行环境感知与路径规划的任务。      |
| **Probability of Detection (PoD)**                 | **检测概率**                  | 主动搜索阶段用于表征“在某个视角/位置发现目标”的概率（用于驱动信息获取）。     |
| **Sim-to-Real Transfer**                            | **虚实迁移 / 仿真到真机迁移** | 将在仿真环境中训练或调优的算法部署到真实物理机器人的过程。                     |
| **Zero-shot**                                       | **零样本**                    | 模型在未见过的环境或类别上直接应用的能力，无需额外训练。                       |

## 2. 系统方法与算法 (Methodology & Algorithms)

| 英文/缩写 (En)                                    | 中文翻译 (Cn)                      | 定义/备注                                                                                                  |
| :------------------------------------------------ | :--------------------------------- | :--------------------------------------------------------------------------------------------------------- |
| **Acoustic Hallucinated Momentum**                | **声学虚拟动量**（亦称 声学幻觉动量） | 本文提出的核心机制：基于声源 DoA 在遮挡区投影“虚拟风险/动量”，引导规划器在拐角处提前做宽转弯以降低 NLOS 碰撞风险。 |
| **Direction-of-Arrival (DoA)**                    | **波达方向 / 到达方向**            | 声音信号传到麦克风阵列的方向角度。                                                                         |
| **Sound Event Localization and Detection (SELD)** | **声学事件定位与检测**             | 识别声音类别（如脚步声）并确定其空间位置的任务。                                                           |
| **Active Search**                                 | **主动搜索**                       | 机器人主动改变姿态或路径以获取更多信息（如寻找声源）的策略。                                               |
| **Search--Then--Go**                               | **先搜后走（闭环搜索-到达）**      | 一种闭环策略：目标未被视觉确认时执行主动搜索；一旦视觉锚定目标，切换为风险感知的目标导向导航。              |
| **(Un)Anchored Target/Entity**                    | **（未）锚定目标/实体**            | 
“锚定”指音频观测与视觉实体在时空与语义上完成关联；未锚定表示仅有音频线索（DoA）但缺少可靠的视觉实体对应。 |
| **De-anchoring**                                  | **去锚定（解除锚定）**              | 当视觉实体长时间离开视野但音频持续存在时，解除音视关联，回退到 NLOS 风险注入与主动搜索逻辑。                 |
| **Information-Theoretic Utility**                 | **信息论效用**                     | 基于信息增益（如互信息）来评估动作价值的指标，用于指导主动搜索。                                           |
| **Next-Best-View (NBV)**                          | **下一最佳视角**                   | 在主动感知中，选择能最大化信息获取的下一个观测位置或角度。                                                 |
| **Target Belief Map**                             | **目标置信图 / 信念地图**           | 对目标可能位置的栅格概率分布；结合“音频正证据 + 视觉负证据”迭代更新，用于 NBV 选择。                          |
| **Social Force Model (SFM)**                      | **社交力模型**                     | 用虚拟力（引力/斥力）描述人群运动交互的经典模型，常用于模拟行人行为或设计代价函数。                        |
| **Fast Marching Method (FMM)**                    | **快速行进法**                     | 一种数值算法，常用于在栅格地图上计算从起点到目标点的最短路径距离场。                                       |
| **Cost Map**                                      | **风险场 / 代价地图**              | 用于表示环境通行代价的栅格图，数值越高代表碰撞风险或社交代价越高。                                         |
| **Cross-Modal Attention**                         | **跨模态注意力**                   | 在神经网络中用于融合视觉和听觉特征的机制，使模型关注不同模态间的关联。                                     |

## 3. 实验指标 (Metrics)

| 英文/缩写 (En)                            | 中文翻译 (Cn)          | 定义/备注                                                                                                                                                         |
| :---------------------------------------- | :--------------------- | :---------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Success Rate (SR)**                     | **成功率**             | 机器人成功导航至目标位置的测试次数百分比。                                                                                                                        |
| **Success Weighted by Path Length (SPL)** | **路径长度加权成功率** | 综合考量成功率与路径最优性的指标，公式为 $\frac{1}{N} \sum_{i=1}^{N} S_i \frac{l_i}{\max(p_i, l_i)}$，其中 $S_i$ 为成功标识，$l_i$ 为最短路径，$p_i$ 为实际路径。 |
| **Personal Space Violation (PSV)**        | **个人空间侵犯率**     | 机器人侵入行人 **0.5m** 舒适距离（Comfort Zone）的时间百分比。                                                                                                    |
| **NLOS Collision Rate (NCR)**             | **非视距碰撞率**       | 特指在视觉检测到行人前 **1.0秒** 内发生的碰撞比例，用于衡量对盲区风险的预判失败。                                                                                 |
| **Time to First Sighting (TTFS)**         | **首次发现时间**       | 从任务开始到机器人相机视野首次捕捉到目标声源的时间，用于衡量主动搜索效率 (Search Efficiency)。                                                                    |
| **Search Efficiency (SE)**                | **搜索效率**           | 用于衡量主动搜索阶段效率的指标；本文用 TTFS 作为主要度量（TTFS 越小表示搜索越高效）。                                                                           |
| **Collision Rate**                        | **碰撞率**             | 发生碰撞的测试次数占比。                                                                                                                                          |
| **Path Length (PL)**                      | **路径长度**           | 机器人从起点到终点移动的总距离。                                                                                                                                  |

## 4. 实验设置与基准 (Experimental Setup & Baselines)

| 英文/缩写 (En)        | 中文翻译 (Cn)        | 定义/备注                                                                                                             |
| :-------------------- | :------------------- | :-------------------------------------------------------------------------------------------------------------------- |
| **SoundSpaces 2.0**   | -                    | 用于模拟物理声学环境（如混响、双耳音频）的仿真平台。                                                                  |
| **Habitat-Lab 3.0**   | -                    | 支持动态社会智能体（Dynamic Social Agents）的高保真仿真环境。                                                         |
| **Stretch 3**         | **Stretch 3 机器人** | Hello Robot 公司开发的移动操作机器人，本文用于真机实验。配备 4 麦克风阵列和 RealSense D435i 相机。                    |
| **Falcon**            | -                    | **基准方法 (Vision SOTA)**。基于视觉的社交导航方法 (ICRA 2025)，实验中增加音频 Oracle (提供目标坐标) 以进行公平对比。 |
| **Audio Oracle**      | **音频预言机**       | 用于对照实验的“理想化”信息源：直接提供目标坐标，隔离并检验系统在规划/避障层面的改进贡献。                           |
| **ENMuS3**            | -                    | **基准方法 (AV-Nav SOTA)**。端到端强化学习视听导航方法 (AAAI 2025)，使用双耳音频。                                    |
| **Standard-AV**       | -                    | **基准方法 (Modular Baseline)**。本文的消融版本，仅使用“Go-to-Sound”简单逻辑，无主动搜索和幻觉动量机制。              |
| **Level 1-4 (L1-L4)** | **难度分级**         | 实验场景难度从 L1 (静态无干扰) 到 L4 (动态行人+脚步声+背景嘈杂声) 递增。                                              |
